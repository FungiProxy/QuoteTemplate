BabbittQuoteGenerator v1.0.0 - Portable Edition

This is the portable version of BabbittQuoteGenerator.
No installation required - just run run.bat or BabbittQuoteGenerator.exe directly.

FEATURES:
- No installation required
- Runs from any location
- No registry changes
- No system files modified

USAGE:
1. Extract to any folder
2. Run BabbittQuoteGenerator.exe
3. Application will create local database files

NOTE: This version stores all data locally in the application folder.
