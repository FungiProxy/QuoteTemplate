Babbitt Quote Generator - Portable Version
==========================================

QUICK SETUP
-----------
1. Unzip the downloaded zip file to any folder
2. Double-click "BabbittQuoteGenerator.exe" to run the application
3. No installation required - it's ready to use!

USER GUIDE
----------
For detailed instructions and help, open the "docs/user_guide.html" file in your web browser.

SUPPORT
-------
If you encounter any issues, please contact the development team with:
- Screenshots of any error messages
- Description of what you were trying to do
- Your operating system version

Enjoy testing!
